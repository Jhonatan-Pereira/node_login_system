## Getting Started

### Install
```sh
npm install
```

### Migration CLI

#### Install Dependencie
```sh
npm install knex -g
```

#### Update Database
```sh
knex migrate:latest
```

### Development
```sh
npm start
```

### Endpoint
```sh
http://localhost:4000
```
